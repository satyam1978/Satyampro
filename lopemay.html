About the Project

StreetSupply ‚Äì Smart Ingredient Sourcing for Food Vendors

StreetSupply is a web-based platform designed to revolutionize how local food vendors source fresh ingredients. The platform offers an efficient, transparent, and cost-effective solution by allowing vendors to:

  • Compare prices instantly from multiple suppliers
  • Locate nearby vendors for faster deliveries
  • Verify product quality through ratings and reviews

This project was developed in just 48 hours during the Tutedude Hackathon (25th ‚Äì 27th July) by our team SAGA.

Our Process

1. Ideation & Research

We identified challenges faced by local food vendors ‚Äî high sourcing time, fluctuating prices, and lack of verified suppliers. Our solution focused on a simple, vendor-friendly interface with real-time price comparisons.

2. UI/UX Design

  • Tool Used: Figma
  • Created wireframes and high-fidelity prototypes
  • Designed a clean, modern, mobile-friendly interface
  • Prioritized accessibility with clear typography and high contrast

3. Development

  • Code Editor: Visual Studio Code (VS Code)
  • Technologies: HTML5, CSS3 (Flexbox, Grid, Animations), JavaScript (ES6), LocalStorage
  • Implemented smooth animations and transitions for a premium experience

4. Testing & Refinement

  • Cross-browser testing on Chrome, Edge, and Firefox
  • Responsive testing on multiple devices using Chrome DevTools
  • Performance optimizations for faster load times

Key Features

  • Glass-effect sticky navigation bar with scroll animation
  • Dynamic price comparison box updating in real time
  • Animated section reveals for better user engagement
  • Prototype vendor location mapping system

Outcome

In just two days, we delivered a functional prototype that received positive feedback from hackathon judges for its clarity, usability, and problem-solving approach. The project showcased our teamwork, adaptability, and ability to deliver under tight deadlines.
